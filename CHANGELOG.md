## [1.0.0] - 2026-03-02

### Added
- Complete rewrite as tool-agnostic compliance scanning platform
- Core scanning engine supporting multiple LLM frameworks
- Compliance rule engine for framework evaluation
- Policy configuration system
- Reporting and export functionality
- Integration with popular LLM frameworks (LangChain, CrewAI, OpenAI, etc.)
- GitHub Actions CI/CD workflows
- Comprehensive test coverage
- API documentation and examples
- Community contribution guidelines

### Changed
- Refactored codebase for modularity and extensibility
- Improved performance of compliance checks
- Enhanced error handling and logging

### Fixed
- Resolved compatibility issues with multiple Python versions

### Security
- Implemented secure credential handling
- Added input validation across all APIs
- Established responsible disclosure policy

## Versioning

Starting with version 1.0.0, this project follows Semantic Versioning:
- MAJOR: Breaking changes
- MINOR: New functionality (backwards compatible)
- PATCH: Bug fixes (backwards compatible)
